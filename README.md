# Attack Shark WebDriver

The Attack Shark WebDriver is an essential tool that allows users to interact with Attack Shark's features programmatically. It simplifies the testing and automation processes by providing a direct interface to the application's functionalities.

## Why Udev Rules are Necessary for Mouse Customization

Udev rules are critical for customizing mouse behavior in Linux environments. These rules help in defining how the operating system interacts with hardware devices, thus allowing users to configure device settings, permissions, and other behaviors that can enhance usability and performance. For mouse customization specifically, correct udev rules ensure that the right drivers and settings are applied, letting users achieve tailored mouse behaviors that suit their preferences.
