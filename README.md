# Setup Instructions

## Requirements

- Node.js 14 or higher
- npm 6 or higher

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/demonn-web/attack-shark-r5-udev-rules.git
   cd attack-shark-r5-udev-rules
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

- To start the application, run:
  ```bash
  npm start
  ```

- For testing, run:
  ```bash
  npm test
  ```
